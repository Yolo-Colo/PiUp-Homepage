Project PiUp

An Over Enginnered Way To Run Some Rasberry Pis

But Why?

I got the idea at the start of quarantine, becuase quarantine really makes you think of silly things to occupi time. The project started out out as this would be fun to try and use up stuff I had from past projects. It slowly turned into what it looks like today.

What Are They Doing?

The Gen1 Pi is running :

  • Pie digit calculations

The Gen2 Pi is running :

  • Munin server for collecting data from both Pis
  • Cron Scrips
  • Nginx Server for external webpages

The Mikrotik Roucter (which is not part of this experiment) is running

  • SMB for a sharerd log and data location
  • NTP Server
  • The Dude for network monitoring
